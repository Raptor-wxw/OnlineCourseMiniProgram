## 介绍

**OnlineCourseMiniProgram是一款大学生网课答案搜题的网课搜题微信小程序，非常好用。可以查询400多门课程的所有题目及其答案，支持查询超星，智慧树，知到，Welearn，四史，马原，毛概，大学mooc，等各种网课，题库不断更新中，非常全面，且支持模糊搜索！！**


----
